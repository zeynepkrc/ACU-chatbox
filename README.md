# ACU AI Chatbot

Backend for the Acıbadem University AI chatbot: Django application, PostgreSQL storage, and responsible public web scraping (main site and Bologna pages).

## Layout

- `docker-compose.yml` — orchestration (completed when the Docker stack is added).
- `.env.example` — environment variable template.
- `webapp/` — Django project root (application code, scraper, templates, static files).

## Next steps
